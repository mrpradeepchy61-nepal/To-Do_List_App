# 📝 To-Do List App

A simple and user-friendly To-Do List web application built using HTML, CSS, and JavaScript. This app helps users manage their daily tasks efficiently with data saved in LocalStorage.

---

## 🚀 Features

- ✅ Add new tasks
- 🗑️ Delete tasks
- 💾 Data saved using LocalStorage (no data loss on refresh)
- 🎨 Clean and responsive UI
- ⚡ Fast and lightweight

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
